* 자료형의 참과 거짓

|------------+---------------------+------------|
| Data Type  | Value               | True/False |
|------------+---------------------+------------|
| String     | "Python"            | T          |
|------------+---------------------+------------|
|            | ""                  | F          |
|------------+---------------------+------------|
| List       | [1,2,3]             | T          |
|------------+---------------------+------------|
|            | []                  | F          |
|------------+---------------------+------------|
| Tuple      | ()                  | F          |
|------------+---------------------+------------|
| Dictionary | {}                  | F          |
|------------+---------------------+------------|
| Number     | number other than 0 | T          |
|------------+---------------------+------------|
|            | 0                   | F          |
|------------+---------------------+------------|
|            | None                | F          |
|------------+---------------------+------------|
